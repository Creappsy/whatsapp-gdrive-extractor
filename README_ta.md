* **அழகான இணைய UI/UX:** பிரித்தெடுத்தல் செயல்முறையின் மூலம் உங்களை எளிதாக வழிநடத்தும் கண்ணாடிமார்பிஸம் கூறுகளுடன் கூடிய நவீன, இருண்ட பயன்முறை நேட்டிவ் இன்டர்ஃபேஸ்.
* **🌍 40 மொழிகள் ஆதரிக்கப்படுகின்றன:** 100% மனித தரமான மொழிபெயர்ப்புகள் உள்ளமைக்கப்பட்டவை. மறுஏற்றம் செய்யாமல் பறக்கும் மொழியை மாற்றவும்!
* **உள்ளூர் மறைகுறியாக்கம் (.crypt14 / .crypt15):** உங்கள் தரவை எங்கும் பதிவேற்றாமல் உங்கள் E2EE ஹாஷ் விசையைப் பயன்படுத்தி உங்கள் SQLite தரவுத்தளங்களைப் பாதுகாப்பாக மறைகுறியாக்க முழு உள்ளூர் குறியாக்க தொகுதி.
* **நேரடி ஆண்ட்ராய்டு பரிமாற்றம் (ADB):** டேஷ்போர்டில் இருந்து ஒரே கிளிக்கில் உங்கள் பதிவிறக்கம் மற்றும் மறைகுறியாக்கப்பட்ட தரவுத்தளத்தை நேரடியாக உங்கள் ஆண்ட்ராய்டு ஃபோனின் சேமிப்பகத்தில் தள்ளுங்கள்.
* **பாதுகாப்பான அங்கீகாரம்:** அதிகபட்ச பாதுகாப்பிற்காக Google OAuth டோக்கன்கள் மற்றும் பயன்பாட்டு கடவுச்சொற்கள் இரண்டையும் ஆதரிக்கிறது.

## 📋 முன்நிபந்தனைகள்

நீங்கள் தொடங்குவதற்கு முன், உங்களிடம் இருப்பதை உறுதிப்படுத்திக் கொள்ளுங்கள்:
1. **Python 3.x** அல்லது **Docker** நிறுவப்பட்டது.
2. வாட்ஸ்அப் நிறுவப்பட்ட மற்றும் கூகுள் டிரைவ் காப்புப்பிரதிகள் இயக்கப்பட்ட ஆண்ட்ராய்டு சாதனம்.
3. உங்கள் Google கணக்குச் சான்றுகள் (அல்லது [ஆப் கடவுச்சொல்](https://myaccount.google.com/apppasswords)).
4. *விரும்பினால்:* உங்கள் சாதனத்தின் Android ஐடி (Google உங்களை வெளியேற்றும் அபாயத்தைக் குறைக்க).

## 🚀 நிறுவல் & பயன்பாடு

### விருப்பம் 1: பைத்தானைப் பயன்படுத்துதல் (UI க்கு பரிந்துரைக்கப்படுகிறது)

1. களஞ்சியத்தை குளோன் செய்யவும்:
   ```பேஷ்
   git குளோன் https://github.com/daferferso/whatsapp-gdrive-extractor.git
   cd whatsapp-gdrive-extractor
   ```

2. தேவையான சார்புகளை நிறுவவும்:
   ```பேஷ்
   pip நிறுவல் -r requirements.txt
   ```

3. இணைய சேவையகத்தை இயக்கவும்:
   ```பேஷ்
   python server.py
   ```
4. நவீன டாஷ்போர்டை அணுக உங்கள் உலாவியைத் திறந்து `http://localhost:5000` க்குச் செல்லவும்!

### விருப்பம் 2: டோக்கரைப் பயன்படுத்துதல்

1. களஞ்சியத்தை குளோன் செய்து அதற்குள் செல்லவும்.
2. டோக்கர் படத்தை உருவாக்கவும்:
   ```பேஷ்
   டாக்கர் உருவாக்கம் . -t whatsapp-gdrive-extractor
   ```
3. டோக்கர் கொள்கலனை இயக்கவும்:
   * **Linux:** `docker run -v $(pwd):/app -p 5000:5000 -it whatsapp-gdrive-extractor`
   * **Windows (PowerShell):** `docker run -v .:/app -p 5000:5000 -it whatsapp-gdrive-extractor`

## 🔑 அங்கீகார வழிகாட்டி

உங்கள் நிலையான Google மின்னஞ்சல் மற்றும் கடவுச்சொல்லைப் பயன்படுத்துவதில் சிக்கல் ஏற்பட்டால், **OAuth டோக்கன்** முறையைப் பயன்படுத்தவும் (இணைய UI இல் படி 1):
1. `https://getandroidapp.com/` அல்லது ஏதேனும் Google உள்நுழைவு போர்ட்டலுக்குச் செல்லவும்.
2. உங்கள் Google கணக்கில் உள்நுழையவும்.
3. டெவலப்பர் கருவிகளைத் திறக்க `F12` ஐ அழுத்தவும்.
4. **பயன்பாடு** -> **குக்கீகள்** என்பதற்குச் செல்லவும்.
5. `oauth_token` ஐக் கண்டறியவும் (இது பொதுவாக `oauth2_4/XXXXXXXXXXXXXXXXX` போல் தெரிகிறது).
6. அதை நகலெடுத்து இணைய UI இல் ஒட்டவும்.
