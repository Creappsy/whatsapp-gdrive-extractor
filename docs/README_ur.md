## 🌟 خصوصیات

* **خوبصورت ویب UI/UX:** شیشے کی شکل کے عناصر کے ساتھ ایک جدید، ڈارک موڈ مقامی انٹرفیس نکالنے کے عمل میں آسانی سے آپ کی رہنمائی کرنے کے لیے۔
* **🌍 40 زبانیں تعاون یافتہ:** 100% انسانی معیار کے ترجمے مقامی طور پر بلٹ ان۔ دوبارہ لوڈ کیے بغیر پرواز پر زبان تبدیل کریں!
* **مقامی ڈکرپشن (.crypt14 / .crypt15):** آپ کا ڈیٹا کہیں بھی اپ لوڈ کیے بغیر آپ کی E2EE ہیش کلید کا استعمال کرتے ہوئے آپ کے SQLite ڈیٹا بیس کو محفوظ طریقے سے ڈیکرپٹ کرنے کے لیے ایک مکمل مقامی کرپٹوگرافی ماڈیول۔
* **ڈائریکٹ اینڈرائیڈ ٹرانسفر (ADB):** ڈیش بورڈ سے ایک کلک کے ساتھ اپنے ڈاؤن لوڈ اور ڈکرپٹ شدہ ڈیٹا بیس کو براہ راست اپنے اینڈرائیڈ فون کے اسٹوریج میں دھکیلیں۔
* **محفوظ توثیق:** زیادہ سے زیادہ سیکیورٹی کے لیے Google OAuth ٹوکنز اور ایپ پاس ورڈز دونوں کو سپورٹ کرتا ہے۔

## 📋 شرائط

شروع کرنے سے پہلے، یقینی بنائیں کہ آپ کے پاس ہے:
1. **Python 3.x** یا **Docker** انسٹال ہے۔
2. ایک اینڈرائیڈ ڈیوائس جس میں WhatsApp انسٹال ہے اور گوگل ڈرائیو بیک اپ فعال ہے۔
3. آپ کے Google اکاؤنٹ کی اسناد (یا [App Password](https://myaccount.google.com/apppasswords))۔
4. *اختیاری:* آپ کے آلے کی اینڈرائیڈ آئی ڈی (گوگل کے آپ کو لاگ آؤٹ کرنے کے خطرے کو کم کرنے کے لیے)۔

## 🚀 انسٹالیشن اور استعمال

### آپشن 1: ازگر کا استعمال (UI کے لیے تجویز کردہ)

1. ذخیرہ کلون کریں:
   ''بش
   گٹ کلون https://github.com/daferferso/whatsapp-gdrive-extractor.git
   cd whatsapp-gdrive-extractor
   ``

2. مطلوبہ انحصار انسٹال کریں:
   ''بش
   pip install -r requirements.txt
   ``

3. ویب سرور چلائیں:
   ''بش
   python server.py
   ``
4. جدید ڈیش بورڈ تک رسائی کے لیے اپنا براؤزر کھولیں اور `http://localhost:5000` پر جائیں!

### آپشن 2: ڈوکر کا استعمال

1. ذخیرہ کو کلون کریں اور اس میں تشریف لے جائیں۔
2. ڈوکر امیج بنائیں:
   ''بش
   ڈاکر کی تعمیر -t whatsapp-gdrive-extractor
   ``
3. ڈوکر کنٹینر چلائیں:
   * **Linux:** `docker run -v $(pwd):/app -p 5000:5000 -it whatsapp-gdrive-extractor`
   * **Windows (PowerShell):** `docker run -v.:/app -p 5000:5000 -it whatsapp-gdrive-extractor`

## 🔑 توثیق گائیڈ

اگر آپ کو اپنے معیاری گوگل ای میل اور پاس ورڈ کے استعمال میں مسائل کا سامنا ہے، تو **OAuth ٹوکن** طریقہ استعمال کریں (ویب UI پر مرحلہ 1):
1. `https://getandroidapp.com/` یا کسی بھی گوگل لاگ ان پورٹل پر جائیں۔
2. اپنے گوگل اکاؤنٹ سے لاگ ان کریں۔
3. ڈویلپر ٹولز کھولنے کے لیے 'F12' دبائیں۔
4. **درخواست** -> **کوکیز** پر جائیں۔
5. `oauth_token` تلاش کریں (یہ عام طور پر `oauth2_4/XXXXXXXXXXXXXXXXXXX` کی طرح لگتا ہے)۔
6. اسے ویب UI میں کاپی اور پیسٹ کریں۔
