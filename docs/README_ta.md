## 🌟 அம்சங்கள்

* **அழகான UI/UX:** கிளாஸ்மார்பிசம் வடிவமைப்பு, இருண்ட பயன்முறை மற்றும் படிப்படியான மறுசீரமைப்பு வழிகாட்டியுடன் கூடிய நவீன டெஸ்க்டாப் இடைமுகம்.
* **🌍 40 ஆதரிக்கப்படும் மொழிகள்:** தரமான சர்வதேசமயமாக்கல் உள்நாட்டில் ஒருங்கிணைக்கப்பட்டது. உடனடியாக மொழிகளை மாற்றவும்.
* **அல்ட்ராலைட் & இன்டிபென்டன்ட்:** **Tauri & Rust** ஐப் பயன்படுத்தி மீண்டும் கட்டப்பட்டது, பூஜ்ஜிய வெளிப்புற சார்புகளுடன் பைனரி அளவை வெறும் **~10MB** ஆக குறைக்கிறது (ஹோஸ்ட் அமைப்பில் பைதான் இயக்க நேரங்கள் அல்லது .NET தேவையில்லை).
* **பாதுகாப்பான கீரிங் சேமிப்பு:** உங்கள் Google OAuth டோக்கன்களை இயக்க முறைமையின் நேட்டிவ் செக்யூரிட்டி கீரிங்கில் நேரடியாகச் சேமிக்கிறது (விண்டோஸில் நற்சான்றிதழ் மேலாளர், மேகோஸில் கீச்செயின்).
* **உள்ளூர் E2EE மறைகுறியாக்கம் (.crypt14 / .crypt15):** ரஸ்ட்டைப் பயன்படுத்தி அதிவேக, பாதுகாப்பான மற்றும் முற்றிலும் உள்ளூர் AES-256-GCM வாட்ஸ்அப் தரவுத்தள மறைகுறியாக்கம்.
* **தொலைபேசிக்கு நேரடி பரிமாற்றம் (ADB):** உங்கள் டீக்ரிப்ட் செய்யப்பட்ட தரவுத்தளங்களை ஒரே கிளிக்கில் நேரடியாக உங்கள் ஆண்ட்ராய்டு ஃபோனுக்கு மீட்டமைத்து, சிஸ்டம் ஏடிபியை நேட்டிவ் முறையில் அழைக்கவும்.

## 🚀 செயல்படுத்தல் மற்றும் மேம்பாடு

### பயன்பாட்டைத் தொடங்குதல்
வெறுமனே இருமுறை கிளிக் செய்யவும் [run.bat](../run.bat) திட்டத்தின் மூலத்தில் கோப்பு.
இது தொகுக்கப்பட்ட உற்பத்தி இயங்கக்கூடியதை (`app.exe`) உடனடியாக திறக்கும் அல்லது தொகுக்கப்படாவிட்டால் டெவலப்மென்ட் பயன்முறைக்கு திரும்பும்.

### டெவலப்மெண்ட் பயன்முறையில் இயங்குகிறது
ஹாட்-ரீலோடிங் மூலம் டெவலப்மெண்ட் பயன்முறையைத் தொடங்க:
```bash
npx tauri dev
```

### தொகுத்தல் மற்றும் தொகுத்தல்
சுயமாக, கையொப்பமிடப்பட்ட உற்பத்தி நிறுவி தொகுப்புகளை உருவாக்க (MSI, EXE அமைப்பு):
```bash
npx tauri build
```
இதன் விளைவாக நிறுவிகள் `src-tauri/target/release/bundle/` இல் அமைந்திருக்கும்.

பார்க்கவும் [DISTRIBUTION.md](../DISTRIBUTION.md) பேக்கேஜிங் மற்றும் கையொப்பமிடுதல் பற்றிய கூடுதல் விவரங்களுக்கு.
