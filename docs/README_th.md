## 🌟 คุณสมบัติ

* **Web UI/UX ที่สวยงาม:** อินเทอร์เฟซเนทีฟโหมดมืดที่ทันสมัยพร้อมองค์ประกอบ glassmorphism ที่จะแนะนำคุณตลอดกระบวนการแยกข้อมูลได้อย่างง่ายดาย
* **🌍 รองรับ 40 ภาษา:** การแปลคุณภาพระดับมนุษย์ 100% ในตัว เปลี่ยนภาษาได้ทันทีโดยไม่ต้องโหลดซ้ำ!
* **การถอดรหัสในเครื่อง (.crypt14 / .crypt15):** โมดูลการเข้ารหัสในเครื่องอย่างสมบูรณ์เพื่อถอดรหัสฐานข้อมูล SQLite ของคุณอย่างปลอดภัยโดยใช้คีย์แฮช E2EE โดยไม่ต้องอัปโหลดข้อมูลของคุณทุกที่
* **Direct Android Transfer (ADB):** พุชฐานข้อมูลที่ดาวน์โหลดและถอดรหัสของคุณไปยังที่จัดเก็บข้อมูลในโทรศัพท์ Android ของคุณโดยตรงด้วยการคลิกเพียงครั้งเดียวจากแดชบอร์ด
* **การตรวจสอบสิทธิ์อย่างปลอดภัย:** รองรับทั้งโทเค็น Google OAuth และรหัสผ่านสำหรับแอปเพื่อความปลอดภัยสูงสุด

## 📋 ข้อกำหนดเบื้องต้น

ก่อนที่คุณจะเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณมี:
1. ติดตั้ง **Python 3.x** หรือ **Docker** แล้ว
2. อุปกรณ์ Android ที่ติดตั้ง WhatsApp และเปิดใช้งานการสำรองข้อมูล Google Drive
3. ข้อมูลรับรองบัญชี Google ของคุณ (หรือ [รหัสผ่านสำหรับแอป](https://myaccount.google.com/apppasswords))
4. *ทางเลือก:* รหัส Android ของอุปกรณ์ของคุณ (เพื่อลดความเสี่ยงที่ Google จะนำคุณออกจากระบบ)

## 🚀 การติดตั้งและการใช้งาน

### ตัวเลือก 1: การใช้ Python (แนะนำสำหรับ UI)

1. โคลนพื้นที่เก็บข้อมูล:
   ``` ทุบตี
   คอมไพล์โคลน https://github.com/daferferso/whatsapp-gdrive-extractor.git
   ซีดี whatsapp-gdrive-extractor
   ```

2. ติดตั้งการพึ่งพาที่จำเป็น:
   ``` ทุบตี
   pip ติดตั้ง -r ข้อกำหนด.txt
   ```

3. เรียกใช้เว็บเซิร์ฟเวอร์:
   ``` ทุบตี
   หลามเซิร์ฟเวอร์.py
   ```
4. เปิดเบราว์เซอร์ของคุณแล้วไปที่ `http://localhost:5000` เพื่อเข้าถึงแดชบอร์ดที่ทันสมัย!

### ตัวเลือก 2: การใช้นักเทียบท่า

1. โคลนพื้นที่เก็บข้อมูลและนำทางเข้าไป
2. สร้างอิมเมจนักเทียบท่า:
   ``` ทุบตี
   สร้างนักเทียบท่า -t whatsapp-gdrive-แยก
   ```
3. เรียกใช้คอนเทนเนอร์นักเทียบท่า:
   * **Linux:** `นักเทียบท่า run -v $(pwd):/app -p 5000:5000 -it whatsapp-gdrive-extractor`
   * **Windows (PowerShell):** `นักเทียบท่า run -v .:/app -p 5000:5000 -it whatsapp-gdrive-extractor`

## 🔑 คู่มือการรับรองความถูกต้อง

หากคุณประสบปัญหาในการใช้อีเมลและรหัสผ่านมาตรฐานของ Google ให้ใช้วิธี **OAuth Token** (ขั้นตอนที่ 1 บน UI ของเว็บ):
1. ไปที่ `https://getandroidapp.com/` หรือพอร์ทัลเข้าสู่ระบบ Google ใดก็ได้
2. เข้าสู่ระบบด้วยบัญชี Google ของคุณ
3. กด `F12` เพื่อเปิดเครื่องมือสำหรับนักพัฒนา
4. ไปที่ **แอปพลิเคชัน** -> **คุกกี้**
5. ค้นหา `oauth_token` (โดยปกติจะมีลักษณะเช่นนี้ `oauth2_4/XXXXXXXXXXXXXXXX`)
6. คัดลอกและวางลงใน Web UI
