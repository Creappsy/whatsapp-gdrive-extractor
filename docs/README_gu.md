## 🌟 લક્ષણો

* **સુંદર UI/UX:** ગ્લાસમોર્ફિઝમ ડિઝાઇન, ડાર્ક મોડ અને સ્ટેપ-બાય-સ્ટેપ રિસ્ટોરેશન ગાઇડ સાથેનું આધુનિક ડેસ્કટોપ ઇન્ટરફેસ.
* **🌍 40 સમર્થિત ભાષાઓ:** ગુણવત્તા આંતરરાષ્ટ્રીયકરણ નેટીવલી સંકલિત. તરત જ ભાષાઓ સ્વિચ કરો.
* **અલ્ટ્રાલાઇટ અને સ્વતંત્ર:** **ટૌરી અને રસ્ટ** નો ઉપયોગ કરીને પુનઃબીલ્ડ, શૂન્ય બાહ્ય નિર્ભરતા સાથે દ્વિસંગી કદને માત્ર **~10MB** સુધી ઘટાડીને (હોસ્ટ સિસ્ટમ પર પાયથોન રનટાઈમ અથવા .NET જરૂરી નથી).
* **સુરક્ષિત કીરીંગ સ્ટોરેજ:** તમારા Google OAuth ટોકન્સને સીધા જ ઑપરેટિંગ સિસ્ટમની નેટિવ સિક્યોર કીરિંગમાં સાચવે છે (વિન્ડોઝમાં ઓળખપત્ર વ્યવસ્થાપક, macOSમાં કીચેન).
* **સ્થાનિક E2EE ડિક્રિપ્શન (.crypt14 / .crypt15):** રસ્ટનો ઉપયોગ કરીને હાઇ-સ્પીડ, સુરક્ષિત અને સંપૂર્ણ સ્થાનિક AES-256-GCM WhatsApp ડેટાબેઝ ડિક્રિપ્શન.
* **ફોન પર ડાયરેક્ટ ટ્રાન્સફર (ADB):** તમારા ડિક્રિપ્ટેડ ડેટાબેસેસને તમારા એન્ડ્રોઇડ ફોન પર સીધા જ એક ક્લિકથી પુનઃસ્થાપિત કરો, સિસ્ટમ ADB ને મૂળ રીતે કૉલ કરો.

## 🚀 અમલ અને વિકાસ

### એપ્લિકેશન લોન્ચ કરી રહ્યા છીએ
ફક્ત ડબલ-ક્લિક કરો [run.bat](../run.bat) પ્રોજેક્ટના રુટ પર ફાઇલ.
જો તે કમ્પાઈલ કરેલ ઉત્પાદન એક્ઝિક્યુટેબલ (`app.exe`) અથવા ડેવલપમેન્ટ મોડમાં ફોલબેકને તરત જ ખોલશે.

### વિકાસ મોડમાં ચાલી રહ્યું છે
હોટ-રીલોડિંગ સાથે વિકાસ મોડ શરૂ કરવા માટે:
```bash
npx tauri dev
```

### કમ્પાઇલિંગ અને બંડલિંગ
સ્વ-સમાયેલ, હસ્તાક્ષરિત ઉત્પાદન ઇન્સ્ટોલર પેકેજો (MSI, EXE સેટઅપ) જનરેટ કરવા માટે:
```bash
npx tauri build
```
પરિણામી ઇન્સ્ટોલર્સ `src-tauri/target/release/bundle/` માં સ્થિત હશે.

જુઓ [DISTRIBUTION.md](../DISTRIBUTION.md) પેકેજિંગ અને સહી કરવા વિશે વધુ વિગતો માટે.
