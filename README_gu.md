## 🌟 સુવિધાઓ

* **સુંદર વેબ UI/UX:** એક આધુનિક, ડાર્ક-મોડ નેટીવ ઇન્ટરફેસ જેમાં ગ્લાસમોર્ફિઝમ તત્વો છે જે તમને નિષ્કર્ષણ પ્રક્રિયામાં સરળતાથી માર્ગદર્શન આપે છે.
* **🌍 40 ભાષાઓ સપોર્ટેડ છે:** 100% માનવ-ગુણવત્તાવાળા અનુવાદો મૂળ રીતે બિલ્ટ-ઇન છે. ફરીથી લોડ કર્યા વિના ફ્લાયમાં ભાષા બદલો!
* **સ્થાનિક ડિક્રિપ્શન (.crypt14 / .crypt15):** તમારો ડેટા ગમે ત્યાં અપલોડ કર્યા વિના તમારી E2EE હેશ કીનો ઉપયોગ કરીને તમારા SQLite ડેટાબેસેસને સુરક્ષિત રીતે ડિક્રિપ્ટ કરવા માટે સંપૂર્ણ સ્થાનિક ક્રિપ્ટોગ્રાફી મોડ્યુલ.
* **ડાયરેક્ટ એન્ડ્રોઇડ ટ્રાન્સફર (ADB):** તમારા ડાઉનલોડ કરેલ અને ડિક્રિપ્ટેડ ડેટાબેસને તમારા એન્ડ્રોઇડ ફોનના સ્ટોરેજમાં સીધા જ ડેશબોર્ડથી એક ક્લિક વડે દબાણ કરો.
* **સુરક્ષિત પ્રમાણીકરણ:** મહત્તમ સુરક્ષા માટે Google OAuth ટોકન્સ અને એપ્લિકેશન પાસવર્ડ બંનેને સપોર્ટ કરે છે.

## 📋 પૂર્વજરૂરીયાતો

તમે પ્રારંભ કરો તે પહેલાં, ખાતરી કરો કે તમારી પાસે છે:
1. **Python 3.x** અથવા **ડોકર** ઇન્સ્ટોલ કરેલું.
2. WhatsApp ઇન્સ્ટોલ કરેલ અને Google ડ્રાઇવ બેકઅપ સક્ષમ કરેલ Android ઉપકરણ.
3. તમારા Google એકાઉન્ટ ઓળખપત્રો (અથવા [એપ પાસવર્ડ](https://myaccount.google.com/apppasswords)).
4. *વૈકલ્પિક:* તમારા ઉપકરણનું Android ID (Google તમને લોગ આઉટ કરવાનું જોખમ ઘટાડવા માટે).

## 🚀 ઇન્સ્ટોલેશન અને ઉપયોગ

### વિકલ્પ 1: પાયથોનનો ઉપયોગ કરવો (UI માટે ભલામણ કરેલ)

1. રીપોઝીટરીને ક્લોન કરો:
   ``બાશ
   git ક્લોન https://github.com/daferferso/whatsapp-gdrive-extractor.git
   cd whatsapp-gdrive-extractor
   ```

2. જરૂરી અવલંબન સ્થાપિત કરો:
   ``બાશ
   pip install -r requirements.txt
   ```

3. વેબ સર્વર ચલાવો:
   ``બાશ
   python server.py
   ```
4. આધુનિક ડેશબોર્ડને ઍક્સેસ કરવા માટે તમારું બ્રાઉઝર ખોલો અને `http://localhost:5000` પર જાઓ!

### વિકલ્પ 2: ડોકરનો ઉપયોગ કરવો

1. રીપોઝીટરીને ક્લોન કરો અને તેમાં નેવિગેટ કરો.
2. ડોકર ઇમેજ બનાવો:
   ``બાશ
   ડોકર બિલ્ડ. -t whatsapp-gdrive-extractor
   ```
3. ડોકર કન્ટેનર ચલાવો:
   * **Linux:** `ડોકર રન -v $(pwd):/app -p 5000:5000 -it whatsapp-gdrive-extractor`
   * **Windows (PowerShell):** `ડોકર રન -v.:/app -p 5000:5000 -it whatsapp-gdrive-extractor`

## 🔑 પ્રમાણીકરણ માર્ગદર્શિકા

જો તમને તમારા પ્રમાણભૂત Google ઈમેઈલ અને પાસવર્ડનો ઉપયોગ કરવામાં સમસ્યા આવે છે, તો **OAuth ટોકન** પદ્ધતિનો ઉપયોગ કરો (વેબ UI પર પગલું 1):
1. `https://getandroidapp.com/` અથવા કોઈપણ Google લોગિન પોર્ટલ પર જાઓ.
2. તમારા Google એકાઉન્ટથી લોગ ઇન કરો.
3. ડેવલપર ટૂલ્સ ખોલવા માટે `F12` દબાવો.
4. **એપ્લિકેશન** -> **કુકીઝ** પર જાઓ.
5. `oauth_token` શોધો (તે સામાન્ય રીતે `oauth2_4/XXXXXXXXXXXXXXXXXXX` જેવું દેખાય છે).
6. તેને વેબ UI માં કૉપિ કરીને પેસ્ટ કરો.
