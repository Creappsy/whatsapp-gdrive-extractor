## 🌟 বৈশিষ্ট্য

* **সুন্দর ওয়েব UI/UX:** একটি আধুনিক, ডার্ক-মোড নেটিভ ইন্টারফেস যাতে গ্লাসমর্ফিজম উপাদান রয়েছে যা আপনাকে সহজে নিষ্কাশন প্রক্রিয়ার মাধ্যমে গাইড করতে পারে।
* **🌍 40টি ভাষা সমর্থিত:** 100% মানব-মানের অনুবাদ নেটিভলি বিল্ট-ইন। রিলোড না করে অন-দ্য-ফ্লাই ভাষা পরিবর্তন করুন!
* **স্থানীয় ডিক্রিপশন (.crypt14 / .crypt15):** আপনার ডেটা কোথাও আপলোড না করেই আপনার E2EE হ্যাশ কী ব্যবহার করে আপনার SQLite ডেটাবেসগুলিকে নিরাপদে ডিক্রিপ্ট করার জন্য একটি সম্পূর্ণ স্থানীয় ক্রিপ্টোগ্রাফি মডিউল।
* **ডাইরেক্ট অ্যান্ড্রয়েড ট্রান্সফার (ADB):** ড্যাশবোর্ড থেকে এক ক্লিকে আপনার ডাউনলোড করা এবং ডিক্রিপ্ট করা ডাটাবেসকে সরাসরি আপনার অ্যান্ড্রয়েড ফোনের স্টোরেজে পুশ করুন।
* **নিরাপদ প্রমাণীকরণ:** সর্বাধিক নিরাপত্তার জন্য Google OAuth টোকেন এবং অ্যাপ পাসওয়ার্ড উভয়ই সমর্থন করে।

## 📋 পূর্বশর্ত

আপনি শুরু করার আগে, নিশ্চিত করুন যে আপনার আছে:
1. **Python 3.x** বা **ডকার** ইনস্টল করা আছে।
2. হোয়াটসঅ্যাপ ইনস্টল এবং Google ড্রাইভ ব্যাকআপ সক্ষম সহ একটি Android ডিভাইস৷
3. আপনার Google অ্যাকাউন্টের শংসাপত্র (বা একটি [অ্যাপ পাসওয়ার্ড](https://myaccount.google.com/apppasswords))।
4. *ঐচ্ছিক:* আপনার ডিভাইসের অ্যান্ড্রয়েড আইডি (Google আপনাকে লগ আউট করার ঝুঁকি কমাতে)।

## 🚀 ইনস্টলেশন এবং ব্যবহার

### বিকল্প 1: পাইথন ব্যবহার করা (UI এর জন্য প্রস্তাবিত)

1. সংগ্রহস্থল ক্লোন করুন:
   ```বাশ
   git ক্লোন https://github.com/daferferso/whatsapp-gdrive-extractor.git
   cd whatsapp-gdrive-extractor
   ```

2. প্রয়োজনীয় নির্ভরতা ইনস্টল করুন:
   ```বাশ
   pip install -r requirements.txt
   ```

3. ওয়েব সার্ভার চালান:
   ```বাশ
   python server.py
   ```
4. আপনার ব্রাউজার খুলুন এবং আধুনিক ড্যাশবোর্ড অ্যাক্সেস করতে `http://localhost:5000` এ যান!

### বিকল্প 2: ডকার ব্যবহার করা

1. সংগ্রহস্থল ক্লোন করুন এবং এটিতে নেভিগেট করুন।
2. ডকার ইমেজ তৈরি করুন:
   ```বাশ
   ডকার বিল্ড। -t whatsapp-gdrive-extractor
   ```
3. ডকার কন্টেইনার চালান:
   * **লিনাক্স:** `ডকার রান -v $(pwd):/app -p 5000:5000 -it whatsapp-gdrive-extractor`
   * **উইন্ডোজ (পাওয়ারশেল):** `ডকার রান -ভি.:/অ্যাপ -পি 5000:5000 -এটি হোয়াটসঅ্যাপ-জিড্রাইভ-এক্সট্রাক্টর`

## 🔑 প্রমাণীকরণ গাইড

আপনি যদি আপনার স্ট্যান্ডার্ড Google ইমেল এবং পাসওয়ার্ড ব্যবহার করে সমস্যার সম্মুখীন হন, তাহলে **OAuth টোকেন** পদ্ধতি ব্যবহার করুন (ওয়েব UI-তে ধাপ 1):
1. `https://getandroidapp.com/` বা যেকোনো Google লগইন পোর্টালে যান।
2. আপনার Google অ্যাকাউন্ট দিয়ে লগ ইন করুন।
3. ডেভেলপার টুল খুলতে `F12` টিপুন।
4. **অ্যাপ্লিকেশন** -> **কুকিজ**-এ যান।
5. `oauth_token` খুঁজুন (এটি সাধারণত `oauth2_4/XXXXXXXXXXXXXXXXXXX` এর মতো দেখায়)।
6. কপি করে ওয়েব UI এ পেস্ট করুন।
