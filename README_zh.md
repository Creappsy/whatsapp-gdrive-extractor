## 🌟 功能特点

* **精美的 UI/UX 界面:** 采用现代的暗黑模式和毛玻璃效果。
* **🌍 支持 40 种语言:** 原生集成的 100% 人工级翻译。无需刷新即可切换！
* **本地解密 (.crypt14 / .crypt15):** 完全本地化的密码学模块，使用您的 E2EE 哈希密钥安全解密 SQLite 数据库。
* **直接传输至 Android (ADB):** 在控制面板中一键将解密后的数据库推送到手机存储。
* **安全身份验证:** 支持 Google OAuth 令牌和应用专用密码。

## 🚀 安装与使用

### 选项 1: 使用 Python (推荐用于 UI)
```bash
git clone https://github.com/daferferso/whatsapp-gdrive-extractor.git
cd whatsapp-gdrive-extractor
pip install -r requirements.txt
python server.py
```
在浏览器中打开 `http://localhost:5000`！
